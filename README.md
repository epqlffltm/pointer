##포인터를 이해하기에 앞서서
모든 데이터들은 메모리의 특정한 공간에 저장되어 있다.
자료형에 따라 메모리를 차지하는 공간의 양이 달라지는데,
int라면 8Byte, char이라면, 1Byte를 차지하게 된다.

프로그램은 이 데이터들을 사용하기 위해 각방에 고유한 주소, 즉 address를 붙여 주었다.
```
int a = 1;
```
이렇게 선언하게 되면 4byte의 공간에 1이라는 숫자가 저장되는 것이다.
